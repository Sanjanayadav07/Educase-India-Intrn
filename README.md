# PopX React Application

A responsive React.js application built from the PopX UI design. The application includes a welcome screen, login page, signup page, and account settings page with a clean and modern user interface.

## 🚀 Tech Stack

### Frontend
- React.js
- React Router DOM
- CSS3
- Vite

## ✨ Features

- 🎨 Modern PopX-inspired UI
- 🔐 User Login
- 📝 User Registration
- 👤 Account Settings
- 📱 Responsive Design
- ⚡ Fast Performance with Vite
- ♻️ Reusable Components

## 🛠️ Installation

### Clone the Repository

```bash
git clone https://github.com/Sanjanayadav07/popx-react-app.git
```

### Navigate to Project Directory

```bash
cd popx-react-app
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

The application will start on:

```bash
http://localhost:5173
```
